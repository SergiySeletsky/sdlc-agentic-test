# Use ASP.NET Core for API Development

## Context

The platform requires a RESTful API server for SDLC orchestration with high performance and security.

## Decision

Adopt ASP.NET Core framework for building the API server.

## Rationale

ASP.NET Core provides performance, cross-platform support, and mature security features.

## Consequences

- Developers need proficiency in .NET stack
- Deployment on Linux containers is well supported

## Alternatives

- Node.js with Express
- Java Spring Boot
