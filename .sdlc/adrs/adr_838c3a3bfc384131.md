# Choose PostgreSQL for Persistent Storage

## Context

Need a reliable relational database with strong transactional support and JSON capabilities.

## Decision

Use PostgreSQL as primary persistent storage for platform data.

## Rationale

PostgreSQL offers robust ACID compliance, JSONB support, and scalability.

## Consequences

- Require expertise in PostgreSQL management
- Better support for complex queries and analytics

## Alternatives

- MongoDB
- MySQL
