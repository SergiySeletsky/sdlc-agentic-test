# Perform Real-Time Multi-Distributor Inventory Lookup and Dynamic Substitution - AC

## Statement

AC-1660-1:
Given: A BOM.
When: The system checks distributor inventories.
Then: The system SHALL identify out-of-stock parts and substitute equivalent parts dynamically without user intervention.
Failure Mode: The system SHALL notify the user if no equivalent parts are available and halt order processing.
