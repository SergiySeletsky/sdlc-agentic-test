# Ghost Inventory Prevention Acceptance Criteria

## Statement

AC-5622-1:
Given: Parts selected in BOM.
When: The system performs inventory lookup across distributors.
Then: The system SHALL ensure parts availability or substitute equivalent parts dynamically to avoid out-of-stock errors within 2 minutes.
Failure Mode: The system fails to find available or equivalent parts resulting in user notification for manual selection.

## Rationale

Ensure real-time inventory checks and dynamic substitution prevent order delays.
