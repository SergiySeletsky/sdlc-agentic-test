# Verify Thermal, Power, and Physical Compatibility of BOM Components - AC

## Statement

AC-1947-1:
Given: A generated BOM.
When: The system performs compatibility checks.
Then: The system SHALL flag any thermal, power logic, or physical constraint violations with detailed error messages.
Failure Mode: The system SHALL prevent order submission and notify the user of incompatibility issues if any violations are found.
