# Natural Language to BOM Parsing

## Statement

The system SHALL parse natural language input from the user to generate a complete Bill of Materials (BOM) including chassis, riser cables, rail kits, and PSUs.

## Rationale

To enable users to input configuration requests in natural language and receive a comprehensive BOM.
