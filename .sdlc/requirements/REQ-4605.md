# Reduce Server/Rack Configuration Time from 3 Days to 3 Minutes - AC

## Statement

AC-1620-1:
Given: A user initiates a server/rack configuration.
When: The system completes the process.
Then: The total elapsed time SHALL be less than or equal to 3 minutes.
Failure Mode: The system SHALL log performance issues and alert support if processing exceeds 3 minutes.
